## 🎯 功能特色

### 🌟 多版本體驗

#### V2 版本（經典版）
- 💗 **互動愛心** - 點擊中央大愛心觸發動畫
- ✨ **粒子爆炸** - 12 個愛心粒子向四周散開
- 💌 **愛的訊息** - "I love You" 文字淡入淡出
- 🌊 **漣漪效果** - 背景漣漪擴散動畫
- 💓 **呼吸動畫** - 愛心持續呼吸，象徵永恆的愛

#### V3 版本（三階段互動）
- 🎨 **階段 0：散布** - 12 個小愛心隨機散布全螢幕，帶呼吸動畫
- 💫 **階段 1：匯聚** - 點擊後，小愛心以 1 秒時間匯聚成大愛心，帶夢幻拖尾效果
- 💥 **階段 2：爆炸** - 再次點擊，粒子爆炸、訊息顯示、自動重置
- 🔄 **無限循環** - 可重複體驗，每次都有新的隨機排列
- 💬 **動態提示** - 毛玻璃效果的提示文字，引導互動

#### V4/V5 版本（時間軸模式）⭐ 最新
- 📅 **垂直時間軸** - 展示我們一起的重要時刻，5個時間點均勻分布
- 💑 **回憶展示** - 每個時間點包含日期、標題、照片和描述
- 🎯 **精確動畫** - 大愛心縮小並精確移動到當前版本的時間點
- 🔒 **漸進式開放** - 已開放的時間點可點擊查看，未開放的顯示「敬請期待」
- 💬 **毛玻璃對話框** - 現代美學的資訊展示方式
- 🏷️ **智慧標籤** - 啟用的時間點顯示日期，禁用的顯示「敬請期待」
- 🔄 **完整循環** - 點擊時間軸背景可重新開始體驗
- 📱 **響應式時間軸** - 完美適配所有裝置尺寸


### 🎨 設計亮點

- 🌙 **自動深色模式** - 根據系統設定自動切換
- 📱 **響應式設計** - 完美支援桌面、平板、手機
- ♿ **無障礙支援** - 鍵盤導航、螢幕閱讀器友善
- ⚡ **流暢動畫** - 極簡風格 + Apple 美學
- 🎭 **粉色漸層** - 溫柔的粉色主題（#ff6b9d → #c23866）

## 🚀 快速開始

### 方法 1：直接開啟
```bash
# 使用瀏覽器開啟 index.html
open index.html
```

### 方法 2：本地伺服器（推薦）
```bash
# 使用 Python 啟動簡易伺服器
python3 -m http.server 8000

# 或使用 Node.js (如有安裝 http-server)
npx http-server -p 8000

# 然後在瀏覽器訪問
# http://localhost:8000
```

### 切換版本
```bash
# 切換到 V2 版本
git checkout v2

# 切換到 V3 版本（最新）
git checkout v3

# 切換到主分支（預設 V2）
git checkout main
```
